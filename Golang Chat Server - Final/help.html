Summary of project features

Name field can only write once, and it become read only once first message or command sent

No empty name or empty message allowed

Unique names

Direct message with another user (Private message)

One button to create a new channel

Name list in the To drop-down of ALL connected users is updated automatically when

First connecting to the server

A user connectes or disconnectes

Channel list in the Channel dropdown of ALL connected users is updated automatically when

First time connecting to the server

A new channel is created by a connected client

Usage

Send a message

Enter your username in the Name field (this is only necessary if the name field is not already filled)

Type a message in the Message field

Click the Send button

Send a direct message to another user

Enter your username in the Name field (this is only necessary if the name field is not already filled)

Type a message in the Message field

Select a user from the To dropdown box

Click the Send button

Send a direct message to another user with a command

Enter your username in the Name field (this is only necessary if the name field is not already filled)

Type the command and message in the Message field, such as send a direct message to Recipient

/[Recipient's username]::[your message]

Click the Send button

Sending a message in a channel

Enter your username in the Name field (this is only necessary if the name field is not already filled)

Join the channel by sending a command (only if you have not already joined the channel) such as

Type /join channel2:: in the Message field and then click the Send button

Type message in the Message field

Select the channel you want to send to from the Channel dropdown box

Click the Send button

Channels

Users can create new channels as many as needed. The channel 1 is the default public channel, which is created automatically when a client connectes. Everyone is in channel 1 by default

Creating a channel

Click the Create Channel button to create a new channel, and the new channel will be shown to all connected clients in the Channel dropdown box automatically

Joining a channel

Join a channel by send command to server, such as /join channel2:: 

Enter your username in the Name field (this is only necessary if the name field is not already filled)

Type command /join [Channel's Name]:: in the Message field

Click the Send button

Leaving a channel

Leave a channel by sending a command to server, such as /leave channel2:: 

Type the command /leave channel2::  in Message field

Click Send button

Sending a message in a channel

Enter your username in the Name field (this is only necessary if the name field is not already filled)

Join the channel by sending a command (only if you have not already joined the channel) such as

Type /join channel2:: in the Message field and then click the Send button

Type message in the Message field

Select the channel you want to send to from the Channel dropdown box

Click the Send button

Command

The following commands are supported:

Command

Discription

/[Recipient]::[message]

sends [message] to [Recipient] as a private message

/time::

get current time

/join [Channel Name]::

join [Channel Name] (Make sure it is spelled correctly)

/leave [Channel Name]::   

leave [Channel Name] (Make sure it is spelled correctly)

Sending a Command

Enter your username in the Name field (this is only necessary if the name field is not already filled)

Type the command in the Message field

Click the Send button
